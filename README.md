# Patternos

A catalog of software design patterns with structured documentation and metadata.

## Structure

```
patterns/
  behavioral/     # Patterns concerned with communication between objects
  creational/     # Patterns for object creation mechanisms
  structural/     # Patterns for composing objects and classes
```

Each pattern lives in its own folder and contains:
- `README.md` — description, intent, use cases, and code examples
- `pattern.json` — structured metadata (name, category, tags, related patterns)

## Pattern Categories

### Behavioral
Patterns that identify common communication patterns between objects.

### Creational
Patterns that provide object creation mechanisms, increasing flexibility and reuse.

### Structural
Patterns that explain how to assemble objects and classes into larger structures.
